# 數據工程師學習路線

一份以實務為導向的數據工程師學習路線，涵蓋 SQL、Python、ETL、資料倉儲、Spark、Airflow、串流、雲端與專案實作。

這份路線圖適合：
- 想要有明確學習路徑的初學者
- 想轉職數據工程師的分析師或軟體工程師
- 想建立完整知識體系的在職數據工程師

## 什麼是數據工程師？

數據工程師負責建置和維護資料管線、資料平台和資料基礎設施，讓分析、報表、機器學習和業務系統能有效使用可靠的資料。

主要職責包括：
- 建置 ETL/ELT 管線
- 設計資料倉儲模型
- 處理批次與串流資料
- 管理工作流程編排
- 提升資料品質與監控
- 優化雲端環境的效能與成本
