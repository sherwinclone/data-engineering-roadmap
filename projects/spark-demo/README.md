# Spark Demo Spark 示範

A PySpark job that reads, transforms, and writes data.
使用 PySpark 讀取、轉換、寫入資料。

## Goals 目標

- Run a Spark job locally 在本地執行 Spark 作業
- Practice DataFrame API 練習 DataFrame API
- Understand partitioning and performance 理解分區與效能

## Steps 步驟

1. Read a CSV or Parquet file 讀取 CSV 或 Parquet 檔案
2. Apply transformations (filter, group, aggregate) 套用轉換（篩選、分組、聚合）
3. Write output as Parquet 輸出為 Parquet 格式
4. Check the Spark UI for performance 透過 Spark UI 檢視效能

## Tech Stack 技術棧

- PySpark
- Python 3
- Docker (optional)
