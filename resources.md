# 推薦資源

## 書籍

| 書名 | 適合階段 | 備註 |
|------|----------|------|
| Fundamentals of Data Engineering | 入門～中階 | DE 必讀，觀念完整 |
| Designing Data-Intensive Applications | 中階～進階 | 系統設計聖經 |
| The Data Warehouse Toolkit | 中階 | Kimball 維度建模 |
| Spark: The Definitive Guide | 中階 | Spark 完整指南 |
| Streaming Systems | 進階 | 串流處理深入理解 |
